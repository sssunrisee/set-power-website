Добро пожаловать в SET

Smart Engineering Technologies (SET) — это международная инженерно-технологическая компания, которая помогает предприятиям внедрять современные решения, повышать эффективность и ускорять цифровую трансформацию.

Мы создаём инженерные решения, объединяющие знания, технологии и ответственность ради устойчивого развития энергетики, промышленности и общества.

Наши направления

SET Energy

Цифровая энергетика, Smart Grids, цифровые подстанции, SCADA, EMS, DMS

SET Oil & Gas

Smart Wells, цифровые месторождения, технологии повышения нефтеотдачи

SET AI

Искусственный интеллект, машинное обучение, предиктивная аналитика

SET Automation

Промышленная автоматизация, IoT, роботизация

SET Digital Twin

Цифровые двойники, моделирование, оптимизация жизненного цикла

SET Academy

Образовательные программы, подготовка специалистов
